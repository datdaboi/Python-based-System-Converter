# Number System Converter

</div>

A simple Python-based CLI tool that converts decimal (DEC) numbers into **binary (BIN)**, **hexadecimal (HEX)**, and **octal (OCT)** formats.  
Built with Python and packaged into a standalone `.exe` using [PyInstaller](https://pyinstaller.org).

## Features
- Input any decimal number and instantly get:
  - Binary representation
  - Hexadecimal representation
  - Octal representation
- Works as a Python script (`system_converter.py`) or as a standalone Windows executable (`system_converter.exe`).

## How to Run
Run the executable in `.\dist\system_converter.py`.
